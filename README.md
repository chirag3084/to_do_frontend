# TodoApp Frontend

A modern, responsive task management application built with React, TypeScript, and Vite.

## Documentation

- **[Routes Documentation](./ROUTES.md)** - Complete guide to all application routes and navigation structure

## Technology Stack

- **Framework**: React 18.3.1
- **Router**: React Router DOM v6.28.0
- **Build Tool**: Vite 5.4.10
- **Language**: TypeScript 5.6.2
- **Styling**: Tailwind CSS
- **UI Components**: Radix UI
- **State Management**: Zustand + TanStack Query
- **Form Handling**: React Hook Form + Zod

## Quick Start

### Install Dependencies
```bash
npm install
```

### Run Development Server
```bash
npm run dev
```

### Build for Production
```bash
npm run build
```

### Preview Production Build
```bash
npm run preview
```

### Lint Code
```bash
npm run lint
```

## Application Routes

The application has two main routes:

- **`/`** - Landing page with sign-in and account creation
- **`/dashboard`** - Main dashboard for task management

For detailed information about all routes, see [ROUTES.md](./ROUTES.md).
